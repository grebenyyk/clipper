# Icons

Place your app icons here:

- `icon.png` - Main tray icon (16x16, 32x32 for retina)
- `icon.icns` - macOS app icon
- `icon.ico` - Windows app icon
- `32x32.png`, `128x128.png`, `128x128@2x.png` - Various sizes for the bundle

You can generate icons from a PNG using the Tauri CLI:
```bash
cargo tauri icon /path/to/source.png
```
